A partir de este punto el objetivo es calcular el promedio, varianza y correlacion entre los valores de X y Y de cada grupo de datos del conjunto de datos. Antes de continuar reflexione como haria para tener una tabla como esta:

# Aqui va la tabla

A continuacion se describe como:

Cree un subconjunto para cada valor de Set

uno <- subset(anscombe, Set=="1")
dos <- subset(anscombe, Set=="2")
tres <- subset(anscombe, Set=="3")
cuatro <- subset(anscombe, Set=="4")

Calcule el promedio, varianza, y correlacion para cada objeto y guardelos en una dataframe

XProm <- mean(uno$X)
YProm <- mean (uno$Y)
Xvar <- var(uno$X)
Yvar <- var(uno$Y)
Corr <- cor(uno$X, uno$Y)
SummaryStats <- data.frame(XProm, YProm, Xvar, Yvar, Corr)
SummaryStats

##   XProm    YProm Xvar     Yvar      Corr
## 1     9 7.500909   11 4.127269 0.8164205

Puede repetir el proceso para cada subconjunto o para simplificar puede crear una funcion que reciba un subconjunto y el dataframe SummaryStats y lo devuelva con los valores adicionados. La funcion creada para este ejemplo se llama calc_Stats()

SummaryStats <- calc_Stats(dos,SummaryStats)
SummaryStats <- calc_Stats(tres,SummaryStats)
SummaryStats <- calc_Stats(cuatro,SummaryStats)

El resultado debe verse asi:

Visualizacion del conjunto de datos

Veamos los conjuntos graficamente usando la libreria ggplot2. Descargue e importe la libreria ggplot2

# omita el signo '#' si ya tiene la libreria descargada
# install.packages("ggplot2")
library(ggplot2)

Usaremos metodos de la libreria ggplot2 para crear facetas y trazar la regresion lineal

plotAns <- ggplot(anscombe,aes(X,Y)) + geom_point() + geom_smooth(method=lm, se=FALSE) + facet_grid(. ~ Set)
# promedio X
plotAns <- plotAns + geom_vline (aes (color = "red", xintercept = SummaryStats[1,1]))
# promedio Y
plotAns <- plotAns + geom_hline (aes (color = "green", yintercept = SummaryStats[1,2]))
